# 登录功能使用指南

## 📱 功能说明

已为应用添加了完整的登录功能，用户可以通过登录页面输入云信 IM 账号和 Token 进行登录。

## ✨ 主要特性

1. **登录页面** - 美观的登录界面，支持账号和 Token 输入
2. **自动登录** - 记住用户登录信息，下次启动自动登录
3. **错误提示** - 登录失败时显示详细的错误信息
4. **状态管理** - 使用 SharedPreferences 本地存储登录状态

## 📂 新增文件

- `lib/src/login/login_page.dart` - 登录页面

## 🔧 修改文件

- `lib/src/home/splash_page.dart` - 启动页面逻辑
  - 检查本地是否有保存的登录信息
  - 如果有则自动登录
  - 如果没有或登录失败则跳转到登录页面

- `pubspec.yaml` - 添加依赖
  - `shared_preferences: ^2.2.2` - 用于本地存储

## 🚀 使用流程

### 1. 首次启动
- 应用启动后会检查是否有保存的登录信息
- 如果没有，会自动跳转到登录页面
- 用户输入账号和 Token 后点击登录

### 2. 登录成功
- 登录信息会自动保存到本地
- 跳转到主页面
- 下次启动时会自动登录

### 3. 登录失败
- 显示详细的错误信息（错误码和描述）
- 用户可以重新输入正确的账号和 Token

### 4. 自动登录
- 应用启动时会尝试使用保存的账号和 Token 自动登录
- 如果自动登录失败（如 Token 过期），会清除保存的信息并跳转到登录页面

## 🔐 获取测试账号

1. 访问云信控制台：https://app.yunxin.163.com/
2. 登录您的账号
3. 在"用户管理"中创建测试账号
4. 获取账号的 `accid` 和 `token`
5. 在登录页面输入这些信息

## 💾 数据存储

登录信息存储在 SharedPreferences 中，包括：
- `is_logged_in` - 登录状态（布尔值）
- `account` - 账号
- `token` - Token
- `last_account` - 最后使用的账号（用于自动填充）

## 🎨 UI 特性

- **Material Design** 风格
- **响应式设计** - 适配不同屏幕尺寸
- **加载状态** - 登录时显示加载动画
- **错误提示** - 友好的错误提示框
- **表单验证** - 输入验证，防止空值提交
- **提示信息** - 显示如何获取账号的提示

## 🔄 登出功能（可选扩展）

如果需要添加登出功能，可以在设置页面添加登出按钮：

```dart
// 清除登录信息
final prefs = await SharedPreferences.getInstance();
await prefs.clear();

// 跳转到登录页面
Navigator.of(context).pushAndRemoveUntil(
  MaterialPageRoute(builder: (context) => const LoginPage()),
  (route) => false,
);
```

## 📝 注意事项

1. **Token 安全性** - Token 存储在本地，请确保应用的安全性
2. **Token 有效期** - Token 可能会过期，需要重新登录
3. **网络状态** - 登录需要网络连接
4. **错误处理** - 已添加完整的错误处理和日志记录

## 🐛 调试

查看日志输出：
- `登录成功: [账号]`
- `登录失败: code=[错误码], details=[错误详情]`
- `尝试自动登录: [账号]`
- `没有登录信息，跳转到登录页面`
